腾讯云开发 · 论坛配置

论坛使用 腾讯云开发 CloudBase 存储用户和帖子数据（免费额度充足）。 点此开通腾讯云开发 →

📖 配置步骤：
1. 登录腾讯云 → 开通云开发 → 创建一个环境（免费模板即可）
2. 在环境详情页复制 环境 ID（格式：env-xxxxxxxxxx）
3. 进入「数据库」→ 创建以下集合：ForumPost、ForumReply、users
4. 将集合权限设为「所有用户可读，仅创建者可读写」
5. 填入环境 ID 保存，刷新 forum.html 后生效
💬 微信登录（可选）：
6. 前往 微信开放平台 → 创建网站应用 → 获取 AppID 和 AppSecret
7. 在 CloudBase 控制台「登录方式」中配置微信开放平台 AppID 和 AppSecret
8. 将网站应用的「授权回调域」填写为你的网站域名
